# Florida Power & Light (FPL) Usage Downloader

Download your Florida Power & Light (FPL) usage and store in a local JSON DB

## Usage

Install NPM dependencies:

```sh
yarn install
```

Create an `.env` based on `.env.sample`:

```sh
cp .env.sample .env
```

Run it:

```sh
yarn start
```
